## 🎨 Customization Examples

### Change Primary Color (60 seconds)
1. Open `static/dashboard-theme.css`
2. Find line 8: `--primary-color: #4F46E5;`
3. Change to your color: `--primary-color: #0056B3;`
4. Also change `--primary-light` and `--primary-dark`
5. Save and refresh

### Apply Pre-Made Theme (2 minutes)
1. Open `QUICK_COLOR_CUSTOMIZATION.md`
2. Pick a theme
3. Copy the color values
4. Paste into `static/dashboard-theme.css`
5. Save and refresh

### Customize Dark Mode Only (5 minutes)
1. Open `static/dashboard-theme.css`
2. Find `[data-theme="dark"]` section (line 48)
3. Change background colors
4. Save and refresh in dark mode
